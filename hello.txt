10.‚Ā† ‚Ā†View Commit History
Detailed log:
Bash

Short log:
Bash

Purpose:
Displays commit history.
11.‚Ā† ‚Ā†Remove File from Staging Area
Bash

Purpose:
Removes file from staging area without deleting it locally.
12.‚Ā† ‚Ā†Create GitHub Repository
Steps:
Login to GitHub
Click New Repository
Repository Name: cloudily
Visibility: Public
Click Create Repository
13.‚Ā† ‚Ā†Connect Local Repository to GitHub
Add Remote Repository:
Bash

Bash

Output:
Plain text

origin  https://github.com/mayur278/cloudily.git (push)
14.‚Ā† ‚Ā†Rename Branch
